# Sign Language Detection (YOLOv11 + ST-GCN)

This project detects sign language gestures in real-time using YOLOv11 for hand detection and ST-GCN for gesture recognition.

## Features
- Real-time webcam detection
- Deep learning models (YOLO + ST-GCN)
- Text-to-speech output

## Setup
pip install -r requirements.txt
python main.py

## Folder Structure
- model/ → trained models
- main.py → main application
