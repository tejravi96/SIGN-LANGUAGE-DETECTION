# Sign Language Detection using YOLOv11 + ST-GCN

This project detects sign language gestures in real-time using YOLOv11 for hand detection and ST-GCN for gesture recognition. The system also converts detected signs into speech using text-to-speech.

## 🚀 Features
- Real-time webcam detection
- YOLOv11 for hand detection
- ST-GCN for gesture classification
- Text-to-speech output

## 🛠️ Tech Stack
- Python
- OpenCV
- Mediapipe
- TensorFlow / Keras
- Ultralytics YOLO

## ▶️ How to Run
pip install -r requirements.txt  
python main.py  

## 📂 Project Structure
- main.py → main application  
- model/ → trained models  
- requirements.txt → dependencies  

## 💡 Use Case
Helps bridge communication gap for hearing and speech impaired people.
